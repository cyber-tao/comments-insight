# Product Overview

Comments Insight is an AI-powered Chrome extension that extracts and analyzes comments from social media platforms.

## Core Purpose

Extract comments from social media posts and provide professional AI-driven sentiment analysis, hot comment identification, and data export capabilities.

## Supported Platforms

- YouTube
- Bilibili
- Weibo
- Douyin
- Twitter/X
- TikTok
- Reddit

## Key Features

- AI-driven comment extraction using DOM analysis
- Hierarchical comment and reply visualization
- Sentiment analysis and hot comment identification
- CSV and Markdown export
- Background task execution with progress tracking
- History tracking with search
- Multi-language support (Chinese/English)
- Privacy-first: all data stored locally

## User Workflow

1. Navigate to supported platform
2. Click extension icon to start extraction
3. AI analyzes page structure and extracts comments
4. View analysis results in tree view
5. Export data or save to history

## Development Mode

The extension distinguishes between development and production modes with detailed logging in development for debugging purposes.
